# Suggested Codex Skills

These are the first repo-specific skills to create in Codex Home when you are ready:

- `imx-retrieval-debug`
- `imx-blueprint-schema`
- `imx-latex-export`
- `imx-auth-billing`
- `imx-data-catalogs`
- `imx-marketing-launch`
- `imx-release-checks`

Each skill should contain:

- scope
- what inputs it expects
- step-by-step workflow
- verification checklist
- what is explicitly out of scope

Keep global product rules in `AGENTS.md`.
Keep repeatable domain procedures in skills.
Keep task-specific decisions in threads and thread briefs.
