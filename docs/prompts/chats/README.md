# Ingeniometrix Chat Prompts

These prompt files are the reusable starting context for the recommended persistent chats in Codex.

Usage:

1. Open a new chat for the area you want to work on.
2. Name the chat using the suggested thread name in the file.
3. Paste the prompt block into the first message.
4. Keep the chat focused on that scope only.

Each prompt is optimized to:

- reduce context bloat
- keep Release 0 scope clean
- avoid mixing unrelated product layers
- make future forking and debugging easier
